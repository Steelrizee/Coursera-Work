Описание проекта
Веб-приложение "Expense Tracker" предназначено для учета и анализа личных финансов пользователей.
Оно позволит пользователям добавлять, редактировать и удалять записи о расходах, а также просматривать статистику своих трат.
Приложение будет включать систему авторизации по стандарту OAuth 2.0
что обеспечит безопасность данных пользователей и упростит процесс входа в систему через сторонние сервисы.
Интерфейс будет интуитивно понятным, что сделает его доступным для пользователей с разным уровнем подготовки.

Функционал приложения

Добавление расходов:
Пользователи смогут добавлять новые записи о расходах с указанием суммы, категории и описания.

Редактирование расходов:
Пользователи смогут изменять существующие записи о расходах.

Удаление расходов:
Пользователи смогут удалять записи, которые больше не нужны.

Просмотр статистики:
Пользователи смогут просматривать общую сумму расходов за определенный период и по категориям.

Фильтрация расходов:
Пользователи смогут фильтровать записи по категориям и датам.

Экспорт данных:
Пользователи смогут экспортировать свои данные о расходах в формате CSV для дальнейшего анализа.

Стек технологий
Frontend:

HTML/CSS: Для разметки и стилизации интерфейса.
JavaScript: Для добавления интерактивности на клиентской стороне.
React.js: Для создания пользовательского интерфейса и управления состоянием приложения.
React Router: Для управления маршрутизацией в приложении.
Backend:

Node.js: Для создания сервера и обработки запросов.
Express.js: Для упрощения работы с маршрутизацией и обработкой HTTP-запросов.
MongoDB: Для хранения данных о расходах и пользователях, включая информацию о пользователях и их расходах.
Passport.js: Для реализации аутентификации через OAuth 2.0.
Дополнительные технологии:

Mongoose: Для работы с MongoDB и упрощения взаимодействия с базой данных.
Axios: Для выполнения HTTP-запросов с клиентской стороны.
Chart.js: Для визуализации статистики расходов в виде графиков.
Bootstrap: Для быстрой стилизации интерфейса и адаптивного дизайна.
